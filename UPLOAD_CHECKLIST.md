# Upload Checklist

1. Create public repo: `axezent-ai-poly-reduce-kernel`.
2. Extract the ZIP.
3. Open the extracted folder.
4. Upload all visible files into GitHub.
5. If `.github/workflows/main.yml` does not upload, create it manually and paste `GITHUB_WORKFLOW_MAIN.yml`.
6. Commit.
7. Wait for GitHub Actions to turn green.
8. Create release `v1.0.0`.
