# Release v1.0.0

Initial public release of Axezent AI Poly-Reduce Kernel.

Included:

- finite 3SAT-to-CLIQUE reduction receipt checker
- Python reference checker
- Rust runtime scaffold
- JSON receipt schema
- passing/failing example receipts
- source audit
- GitHub Actions workflow backup
- claim-boundary and enterprise documentation
- donation information

Truth boundary:

Axezent AI PRK verifies finite receipts under declared deterministic reduction rules. It does not resolve P vs NP or certify global solver correctness.
