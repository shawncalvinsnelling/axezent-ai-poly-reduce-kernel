# Claim Boundary

Axezent AI PRK is a deterministic finite reduction-receipt checker.

Accepted public claim:

> Given a submitted finite 3SAT-to-CLIQUE receipt, Axezent AI PRK verifies whether the receipt obeys the declared deterministic reduction rules.

Rejected overclaims:

- The project does not resolve P vs NP.
- The project does not certify global solver correctness.
- The project does not optimize all reductions.
- The project does not automate all of theoretical computer science.
- The project does not guarantee that every external proof is correct.

Use the phrase:

```text
No claim without a receipt.
```
